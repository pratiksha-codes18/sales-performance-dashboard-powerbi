# 📊 Sales Performance Dashboard (Power BI)

## 📌 Overview
This project is an interactive Power BI dashboard designed to analyze sales performance data. It helps in tracking revenue, profit trends, and customer purchasing behavior.

---

## 🎯 Key Features
- 💰 Total Sales Analysis
- 📦 Quantity Sold Tracking
- 📈 Profit & Loss Insights
- 🧾 Average Order Value (AOV)
- 🌍 Region/State-wise Sales Analysis
- 📊 Category & Sub-category Performance
- 👥 Customer-wise Sales Insights
- 📅 Monthly Sales Trends

## 📂 Dataset Details

- Order ID
- Customer Name
- Product Category & Sub-category
- Sales Amount
- Quantity
- Profit
- Order Date
- Region/State

- Detected loss-making segments

## 🚀 How to Use
1. Download the `.pbix` file
2. Open it in Power BI Desktop
3. Use slicers to interact with the dashboard
